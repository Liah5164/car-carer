# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public GitHub issue
2. Email: muller.aleaume@gmail.com
3. Include a description and steps to reproduce
4. Allow 48 hours for an initial response

We take security seriously. All reports will be acknowledged and investigated promptly.
